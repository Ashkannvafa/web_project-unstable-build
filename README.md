# Royalty Splitter - React Migration

This project has been migrated from a static Vue.js site to a modern React + Vite application.

## Prerequisites

- Node.js (v18 or higher)
- npm (comes with Node.js)

## Getting Started

1.  **Install Dependencies**:
    Open a terminal in this directory and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    Open the URL shown in the terminal (usually `http://localhost:5173`).

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

- **3D Pie Chart**: Interactive visualization using `react-three-fiber`.
- **Modern Stack**: React, TypeScript, Vite, Tailwind CSS.
- **Internationalization**: English, Spanish, French support.
- **Dark/Light Mode**: Fully themable UI.
